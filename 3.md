## 1
```sql
SELECT "id", "pizza_name" FROM menu
UNION
SELECT "id", "name" FROM person 
ORDER BY "id", "pizza_name"
```

## 2
```sql
SELECT name, 'person' AS source FROM person
UNION ALL
SELECT pizza_name, 'pizza' FROM menu 
ORDER BY source, name
```
